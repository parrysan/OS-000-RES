# If You Only Watch One Web Design Video, Make It This

The speaker argues that web design is governed by seven stable laws—not trends—prioritising single conversion goals, copy clarity, hero section effectiveness, words-first design, restraint as premium, speed/SEO as foundation, and the site as an ongoing system. Laws (why rules exist) outlast any 18-month trend cycle and prevent the redesign hamster wheel. The restraint principle and copy-first approach directly echo Phil's 'less is more' operating principle.
