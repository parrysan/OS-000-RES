# obsidian-knowledge-brain: auto-annotating technical decision & bug-fix memory across sessions

A zero-dependency, cross-platform (Claude Code, Cursor, Codex) agent skill that auto-annotates [DECISION] and [ERROR] markers during sessions, classifies them into a MECE atom table with root-cause dedup, and evolves extracted patterns into persistent rules. Unlike Phil's manual /capture workflow, this operates in pre/during/post trigger phases without requiring explicit approval gates. The pattern of structured technical-memory injection (not just raw notes) is a meaningful complement to the Open Memory Vault's curated layer.
