# I asked Claude Code to make me as much money as possible

Claude's default tuning optimises for perceived productivity, not output quality. Four structural upgrades — adversarial idea review via a persona council, Playwright-backed verification loops, session-handoff context hygiene, and parallel sub-agents with a separate evaluator for done conditions — systematically close that gap. Together they shift the operator from builder to decision-maker and reviewer.
