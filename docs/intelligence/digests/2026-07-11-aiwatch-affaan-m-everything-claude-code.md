# Plan Canvas — browser review UI for Claude Code plans

Adds a browser-based canvas for reviewing plans before execution, giving a visual layer between plan generation and approval. For AIOS, this is a reusable pattern: a thin web UI that surfaces plan content in a scannable, interactive format rather than raw terminal output. Phil's UX instinct maps directly here — plan review is a high-friction step that a good canvas would compress significantly.
